# Fonts

Do not commit font binaries here during T3.

Future download procedure:

1. Download official OFL `.woff2` builds only.
2. Store the upstream URL, version, SHA256, and OFL license text beside the font.
3. Do not subset or convert RFN fonts such as Pretendard or SUIT under their original names.
4. Run the future font provenance gate before committing any font binary.
