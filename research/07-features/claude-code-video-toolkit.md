## 이 레포에서 배우지 말 것

1. 영어 전용 자막 정렬을 그대로 복사하지 말 것. `gen_captions.py`는 정규화가 `[a-z0-9']` 중심이고 Whisper도 `language="en"`으로 고정되어 한국어 워드싱크에 부적합하다. 근거: `templates/concept-explainer-short/gen_captions.py:35-37`, `templates/concept-explainer-short/gen_captions.py:103`.

2. 누락 에셋을 최종 렌더에서 placeholder로 조용히 통과시키지 말 것. 프리뷰에는 유용하지만 품질 하드게이트 프로젝트에서는 required asset missing을 실패로 처리해야 한다. 근거: `templates/concept-explainer-short/build.py:119-147`, `templates/concept-explainer-short/build.py:209-211`.

3. 테스트 없는 렌더 스크립트 운영을 답습하지 말 것. 주요 템플릿 `package.json`에 `test`나 `lint`가 없고 render/studio/build 중심이라, 우리 프로젝트에는 schema 검증, 렌더 스냅샷, 오디오/자막 게이트를 별도 CI로 강제해야 한다. 근거: `templates/sprint-review-v2/package.json:5-11`, `templates/product-demo/package.json:5-9`.
