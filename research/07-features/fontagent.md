## 이 레포에서 배우지 말 것

1. 외부 추출 도구를 레포 밖 `.tools` 경로에 강결합하지 말 것. `reference_web.py`와 `reference_image.py`는 Node/Swift 추출기를 특정 경로에서 찾도록 되어 있어 배포 재현성이 약하다. 근거: `fontagent/reference_web.py:12-24`, `fontagent/reference_image.py:18-21`.

2. 대시보드 HTML/CSS/JS와 API, 파일 서빙을 거대한 단일 Python 파일에 몰아넣지 말 것. 테스트와 부분 재렌더 구조가 커질수록 유지보수가 어려워지고, artifact path 직접 서빙도 별도 경계 검증이 필요하다. 근거: `fontagent/http_api.py:14-2000`, `fontagent/http_api.py:2120-2126`.

3. 웹 수집을 사이트별 HTML 정규식에 과도하게 의존하지 말 것. 영상 프로젝트에서는 공식 API, 명시 계약, 저장된 스냅샷 fixtures를 우선하고 정규식 파서는 격리해야 한다. 근거: `fontagent/resolver.py:257-295`, `fontagent/discovery.py:184-223`, `fontagent/noonnu.py:227-247`.
