## 이 레포에서 배우지 말 것

1. 비결정적 렌더 입력 선택: 배경 클립 시작점, 이미지 후보 선택, UI 기본 자산 선택이 `random`에 의존한다. Hyperframes 결정론 렌더에서는 seed와 manifest 고정이 필수다. 근거: `shortGPT/editing_utils/handle_videos.py:40`, `shortGPT/editing_utils/editing_images.py:16`, `gui/asset_components.py:58`.

2. 스키마와 실행기 계약 불일치: `background_music.json`은 `normalize_audio` 액션을 선언하지만 실행기는 `normalize_music`만 처리한다. 이런 drift는 JSON schema와 golden manifest test로 막아야 한다. 근거: `shortGPT/editing_framework/editing_steps/background_music.json:20`, `shortGPT/editing_framework/core_editing_engine.py:176`.

3. 외부 에셋 수집의 보안·라이선스 부재: SSL 검증 비활성화, `shell=True` 실행, asset DB의 `type/url/path/ts` 중심 저장은 라이선스와 provenance 하드게이트로 쓰기 어렵다. 근거: `shortGPT/api_utils/image_api.py:56`, `shortGPT/api_utils/image_api.py:58`, `shortGPT/audio/audio_utils.py:95`, `shortGPT/audio/audio_utils.py:97`, `shortGPT/config/asset_db.py:53`.
