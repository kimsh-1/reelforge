# Stat Briefing

한국인 수면 시간을 2024년 생활시간조사 기준으로 해설하는 데이터 브리핑 쇼케이스다. 프리셋은 `data-journal`을 사용해 FT식 종이 톤, 직접 라벨 차트, 절제된 빨간 강조를 살렸고, 8시간 4분·5년 새 8분 감소·잠 못 이룬 사람 11.9%·ICT 여가 사용 증가를 한 흐름으로 보여 주는 것이 의도다. 근거: [국가데이터처 2024 생활시간조사](https://mods.go.kr/board.es?act=view&bid=220&list_no=437764&mid=a10301060400), [대한민국 정책브리핑](https://www.korea.kr/briefing/policyBriefingView.do?newsId=156721673).
